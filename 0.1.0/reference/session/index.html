    ao3_sync.session.AO3Session

    Bases: BaseSettings

    Session object for AO3

    This class is a wrapper around requests.Session that handles all requests to AO3. By default, it will rate limit requests to 1 request every 5 seconds.

    Attributes:

    Name Type Description
    username str | None

    AO3 username
    password SecretStr | None

    AO3 password
    is_logged_in bool

    Is the session logged in? Defaults to False
    NUM_REQUESTS_PER_SECOND float | int

    Number of requests per second. Defaults to 0.2
